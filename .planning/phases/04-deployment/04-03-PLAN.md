<task type="auto">
  <name>Create Docker Compose configuration</name>
  <files>docker-compose.yml</files>
  <action>
Create `docker-compose.yml` in project root with the following services:

**Services:**

1. **chromadb** (Vector Database):
   - Image: `chromadb/chroma:latest`
   - Ports: `8001:8000` (host:container)
   - Volumes: `chromadb_data:/chroma/chroma`
   - Environment: `IS_PERSISTENT=TRUE`, `CHROMA_SERVER_HOST=0.0.0.0`
   - Health check: `curl -f http://localhost:8000/api/v1/heartbeat`

2. **backend** (FastAPI):
   - Build: `./backend`
   - Ports: `8000:8000`
   - Environment variables from .env file
   - Volumes:
     - `app_data:/app/data`
     - `uploads:/app/uploads`
   - Depends on: chromadb (healthy)
   - Restart: unless-stopped
   - Health check: `curl -f http://localhost:8000/api/health`

3. **frontend** (React + Nginx):
   - Build: `./frontend`
   - Ports: `80:80`
   - Depends on: backend (healthy)
   - Restart: unless-stopped
   - Health check: `curl -f http://localhost:80/`

**Networks:**
- `rag-network` (bridge driver)

**Volumes:**
- `chromadb_data` (persistent vector data)
- `app_data` (SQLite database)
- `uploads` (uploaded documents)

**Configuration Requirements:**
- Use Docker Compose v3.8 format
- Enable health checks for all services
- Set proper restart policies
- Configure logging with limits (max-size: 10m, max-file: 3)
- Use internal DNS for service discovery (backend connects to chromadb:8000)

<task type="auto">
  <name>Test Docker Compose setup</name>
  <files>docker-compose.yml</files>
  <action>
Test the Docker Compose configuration:

1. **Validate compose file**:
```bash
docker-compose config
```

2. **Build all services** (without starting):
```bash
docker-compose build --no-cache
```

3. **Start services in detached mode**:
```bash
docker-compose up -d
```

4. **Wait for services to be healthy** (check status):
```bash
docker-compose ps
```

5. **Test service connectivity**:
```bash
# Test ChromaDB
curl http://localhost:8001/api/v1/heartbeat

# Test Backend
curl http://localhost:8000/api/health

# Test Frontend
curl http://localhost:80/
```

6. **View logs**:
```bash
docker-compose logs --tail=50
```

7. **Stop and clean up**:
```bash
docker-compose down -v
```

Document any issues and ensure all services start and communicate correctly.
