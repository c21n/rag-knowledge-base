  <name>Create frontend Dockerfile</name>
  <files>frontend/Dockerfile, frontend/nginx.conf</files>
  <action>
Create `frontend/Dockerfile` with multi-stage build:

1. **Build stage** (Node.js):
   - Base: `node:20-alpine`
   - Working directory: `/app`
   - Copy package.json and package-lock.json first (layer caching)
   - Run `npm ci --only=production` for dependencies
   - Copy source code
   - Build: `npm run build` (creates dist/ folder)

2. **Production stage** (Nginx):
   - Base: `nginx:alpine`
   - Copy custom nginx.conf to `/etc/nginx/conf.d/default.conf`
   - Copy built files from build stage: `COPY --from=build /app/dist /usr/share/nginx/html`
   - Expose port 80
   - Start nginx

Create `frontend/nginx.conf`:
```nginx
server {
    listen 80;
    server_name localhost;
    root /usr/share/nginx/html;
    index index.html;

    # Gzip compression
    gzip on;
    gzip_vary on;
    gzip_min_length 1024;
    gzip_types text/plain text/css application/json application/javascript text/xml;

    # Handle client-side routing
    location / {
        try_files $uri $uri/ /index.html;
    }

    # Cache static assets
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # Security headers
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;
}
```

Also create `frontend/.dockerignore`:
```
node_modules/
.git/
.gitignore
README.md
dist/
.env
.vite/
```

<task type="auto">
  <name>Test frontend container</name>
  <files>frontend/Dockerfile</files>
  <action>
Test the frontend Docker container:

1. Build the image: `docker build -t rag-frontend:test .`

2. Run the container:
```bash
docker run -d \
  --name rag-frontend-test \
  -p 8080:80 \
  rag-frontend:test
```

3. Wait 3 seconds for nginx startup

4. Test the application:
```bash
curl -f http://localhost:8080/ | head -20
```

5. Check that static assets are served:
```bash
curl -I http://localhost:8080/assets/ 2>&1 | grep -E "(HTTP|Content-Type)"
```

6. Verify gzip is working:
```bash
curl -H "Accept-Encoding: gzip" -I http://localhost:8080/ 2>&1 | grep -i encoding
```

7. Check container logs:
```bash
docker logs rag-frontend-test | tail -10
```

8. Clean up:
```bash
docker stop rag-frontend-test && docker rm rag-frontend-test
```

9. Report image size:
```bash
docker images rag-frontend:test --format "{{.Size}}"
```

Document any issues and ensure the container serves the built React app correctly.
