# Enterprise RAG Knowledge Base System

[![Docker](https://img.shields.io/badge/Docker-ready-blue)](https://www.docker.com/)

**2. Features:**
- Document upload (PDF, DOCX, MD, TXT)
- Intelligent chunking and embedding
- Semantic search with ChromaDB
- AI-powered Q&A with source citations
- Multiple role templates (Technical, HR, Product)
- Conversation history and feedback
- Modern React UI with Ant Design

**3. Architecture:**
Brief description + ASCII or reference to architecture diagram
- Frontend: React + TypeScript + Vite + Ant Design
- Backend: FastAPI + Python 3.11
- Vector DB: ChromaDB
- LLM: OpenAI / Alibaba Bailian API

**4. Prerequisites:**
- Docker 20.10+
- Docker Compose 2.0+
- 4GB RAM minimum
- API keys (OpenAI or Alibaba Bailian)

**5. Quick Start:**
```bash
# Clone repository
git clone <repo-url>
cd rag-knowledge-base

# Run setup script
./scripts/setup.sh  # Linux/Mac
# or
.\scripts\setup.ps1  # Windows

# Start services
docker-compose up -d

# Access application
# Frontend: http://localhost
# API Docs: http://localhost:8000/docs
```

**6. Configuration:**
Explain .env file and key variables:
- OPENAI_API_KEY / API_KEY
- DATABASE_URL
- CHROMA_PERSIST_DIR
- LLM_MODEL
- CHUNK_SIZE / CHUNK_OVERLAP

**7. API Endpoints:**
Table of key endpoints:
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/chat | Submit question |
| GET | /api/documents | List documents |
| POST | /api/documents | Upload document |
| GET | /api/health | Health check |

**8. Development:**
```bash
# Backend
cd backend
pip install -r requirements.txt
uvicorn src.main:app --reload

# Frontend
cd frontend
npm install
npm run dev
```

**9. Troubleshooting:**
Common issues:
- "Connection refused" - Check if Docker is running
- "API key error" - Verify OPENAI_API_KEY in .env
- "Port already in use" - Change ports in docker-compose.yml
- "Permission denied" - Run with sudo or check file permissions

**10. Project Structure:**
```

```
