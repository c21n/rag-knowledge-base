## 验证步骤

### 步骤 1: 启动服务器

```bash
cd backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 步骤 2: 上传测试文档

创建测试文件 `test.txt`：
```
机器学习是人工智能的一个重要分支。它使计算机能够从数据中学习，而无需明确编程。
深度学习是机器学习的一个子集，使用神经网络来处理复杂的数据模式。
```

上传文档：
```bash
curl -X POST -F "file=@test.txt" http://localhost:8000/api/documents?process=true
```

### 步骤 3: 检查处理状态

```bash
# 获取文档 ID 后查询状态
curl http://localhost:8000/api/documents/{id}/status
```

### 步骤 4: 测试搜索

```bash
curl -X POST http://localhost:8000/api/search \
  -H "Content-Type: application/json" \
  -d '{"query": "什么是机器学习", "top_k": 5}'
```

**预期结果：**
- 返回包含 "机器学习" 相关的 chunks
- 每个结果包含 `score` 字段（相似度分数，0-1）
- `total_results` 显示结果数量

### 步骤 5: 验证向量存储持久化

```bash
# 检查 ChromaDB 文件是否存在
ls backend/data/chromadb/

# 查看统计信息
curl http://localhost:8000/api/search/stats
```

## 故障排除

**如果搜索返回 500 错误：**
- 检查 `.env` 文件中的 `OPENAI_API_KEY` 是否有效
- 确认阿里百炼 API 密钥格式正确
- 查看后端日志获取详细错误信息

**如果文档处理失败：**
- 检查文件格式是否支持（PDF, DOCX, MD, TXT）
- 确认文件大小不超过 50MB
- 查看 `backend/uploads/` 目录是否有文件
