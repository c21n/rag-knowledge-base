# Checkpoint 01-05-01: 阿里百炼 API Key 设置

**Plan**: 01-05 (Embedding Generation and Vector Storage)
**Type**: `checkpoint:human-action` (blocking gate)
**Created**: 2026-03-04

---

## Status: ⏳ 等待用户操作

### 您需要做什么

嵌入服务需要阿里百炼的API密钥来生成向量嵌入。

#### 步骤 1: 获取阿里百炼 API Key

1. 访问: https://bailian.console.aliyun.com/cn-beijing/?tab=coding-plan#/efm/detail
2. 登录您的阿里云账号
3. 创建或获取 API Key
4. 复制密钥

#### 步骤 2: 创建 `.env` 文件

在 `backend/.env` 创建文件，内容如下:

```env
OPENAI_API_KEY=sk-your-bailian-api-key-here
OPENAI_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
```

**快速命令 (PowerShell):**
```powershell
echo "OPENAI_API_KEY=sk-your-key-here`nOPENAI_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1" > backend/.env
```

**快速命令 (Bash/Git Bash):**
```bash
printf "OPENAI_API_KEY=sk-your-key-here\nOPENAI_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1\n" > backend/.env
```

#### 步骤 3: 验证文件

确保 `backend/.env` 存在且包含您的密钥:
```bash
cat backend/.env
```

---

## 为什么需要这个

- `EmbeddingService` 使用兼容 OpenAI 格式的阿里百炼 API
- 需要设置 `OPENAI_BASE_URL` 指向阿里百炼的端点
- 没有有效的 API 密钥，嵌入生成将失败

## Status: ⏳ PENDING USER ACTION

### What You Need to Do

The embedding service requires an OpenAI API key to generate vector embeddings.

#### Step 1: Get Your OpenAI API Key

1. Go to: https://platform.openai.com/api-keys
2. Sign in to your OpenAI account
3. Click "Create new secret key"
4. Copy the key (starts with `sk-...`)

#### Step 2: Create the `.env` File

Create a file at `backend/.env` with the following content:

```env
OPENAI_API_KEY=sk-your-actual-api-key-here
```

**Quick command (PowerShell):**
```powershell
echo "OPENAI_API_KEY=sk-your-key-here" > backend/.env
```

**Quick command (Bash/Git Bash):**
```bash
echo "OPENAI_API_KEY=sk-your-key-here" > backend/.env
```

#### Step 3: Verify the File

Ensure `backend/.env` exists and contains your key:
```bash
cat backend/.env
```

---

## Why This Is Required

- `EmbeddingService` uses `langchain_openai.OpenAIEmbeddings` which requires `OPENAI_API_KEY`
- Without a valid API key, embedding generation will fail
- The service will initialize but all embedding calls will return errors
